# 🧠 Neural Network: From Basics to Customer Churn Prediction

This project demonstrates a complete learning journey of neural networks — starting from fundamental concepts, building a neural network from scratch using NumPy, and finally applying it to a real-world customer churn prediction problem.

---

## 🚀 Project Overview

This notebook is divided into three main sections:

### 🔹 1. Basic Neural Network Concepts
- Understanding neurons and layers  
- Activation functions  
- Forward propagation intuition  

---

### 🔹 2. Neural Network from Scratch (NumPy)
- Manual implementation of:
  - Weight initialization  
  - Forward propagation  
  - Loss calculation  
- Understanding how neural networks learn internally  
- Building intuition without relying on libraries  

---

### 🔹 3. Customer Churn Prediction (Real Dataset)
- Data preprocessing and feature handling  
- Building a neural network model using libraries  
- Improving model performance  
- Evaluating accuracy and predictions  

---

## 📊 Dataset

- Customer churn dataset  
- Used to predict whether a customer will leave a service  
- Real-world classification problem  

---

## 🧠 Key Learnings

- Neural networks are function approximators that learn patterns from data  
- Forward propagation transforms input through layers  
- Model performance depends on:
  - Data preprocessing  
  - Network architecture  
  - Hyperparameters
